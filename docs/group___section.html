INIAPI INI_section* INI_section_New ( INI_section *  conf,
const char *  name 
)

Create a new configuration.

Parameters
confCan be null. If this parameter is not null, the newly created configuration is stored in this one (but also returned). Also means directly allocated storage of conf won't be managed by the API. If NULL, memory will be managed by the API. In both cases INI_section_Free has to be called to free memory.
nameName of the configuration (When creating the global one: NULL)
Returns
Returns the new configuration. NULL if error occured.
